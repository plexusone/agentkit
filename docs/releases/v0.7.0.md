# AgentKit v0.7.0 Release Notes

**Release Date:** July 13, 2026

## Highlights

This release adds Kubernetes as a standard deployment provider, enabling cloud-agnostic container deployments across EKS, GKE, AKS, and on-premises clusters.

## New Features

### Kubernetes Provider Support

Added `ProviderKubernetes` constant to the deployment registry for Kubernetes-based deployments:

```go
import (
    "github.com/plexusone/agentkit/deploy"
    _ "github.com/plexusone/agentkit-k8s-pulumi/deploy/providers/kubernetes"
)

// Get Kubernetes provider
provider, err := deploy.GetProviderByName(deploy.ProviderKubernetes, cfg)

// Or set via environment variable
// AGENTKIT_DEPLOY_PROVIDER=kubernetes
provider, err := deploy.GetProvider(cfg)
```

### Cloud-Agnostic Deployments

The Kubernetes provider enables the same deployment code to work across:

| Cloud | Kubernetes Service |
|-------|-------------------|
| AWS | EKS (Elastic Kubernetes Service) |
| GCP | GKE (Google Kubernetes Engine) |
| Azure | AKS (Azure Kubernetes Service) |
| On-prem | Any K8s cluster |

### Provider Implementations

Provider implementations remain in separate modules:

| Module | Provider | Description |
|--------|----------|-------------|
| [agentkit-aws-pulumi](https://github.com/plexusone/agentkit-aws-pulumi) | `lightsail` | AWS Lightsail Container Service |
| [agentkit-k8s-pulumi](https://github.com/plexusone/agentkit-k8s-pulumi) | `kubernetes` | Any Kubernetes cluster |

## Installation

```bash
go get github.com/plexusone/agentkit@v0.7.0
```

For Kubernetes deployments:

```bash
go get github.com/plexusone/agentkit-k8s-pulumi@latest
```
